# Docsify + Markdown + GitHub 使用教程

欢迎阅读本教程。你将学会如何用 **Docsify** 搭建文档站点、用 **Markdown** 编写内容，并通过 **GitHub Pages** 免费发布到互联网。

## 教程简介

本教程面向希望快速搭建个人或团队文档站的读者，不要求有前端构建经验。Docsify 无需编译，只需写好 Markdown 和一份简单的 `index.html`，即可在本地或 GitHub 上浏览。

## 适用人群

- 需要写技术文档、项目说明或知识库的开发者
- 想用 Markdown 写文档并希望有导航、搜索的读者
- 希望零成本托管静态文档站（GitHub Pages）的用户

## 前置条件

在开始前，建议准备好：

| 工具 | 用途 |
|------|------|
| **Node.js**（可选） | 使用 `docsify-cli` 本地预览；若无 Node，可用 Python 等静态服务器 |
| **Git** | 版本管理与推送到 GitHub |
| **GitHub 账号** | 创建仓库并启用 GitHub Pages |
